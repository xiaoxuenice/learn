######## 	grep
-r  递归查找
-i  忽略大小写
-e  正则
-o  只显示匹配的部分
-l 列出符合内容的文件
-n 显示列号
-v 翻转查找
-c 统计行数

[] 里面用 -o -a 外面用 and or
if判断
If  [
-z     字符串为空 
-n 		字符串不为空
-e 		有目录和文件夹
-f		文件
-d		目录
-a 		and
-o 		or

sort -t= -k2n a.txt
